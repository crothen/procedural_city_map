# Procedural City Map Generator

A procedural city generator built with React, TypeScript, and Canvas. It generates organic, grid, and radial city layouts with water features, bridges, and zoning.

**[Live Demo](https://crothen.github.io/procedural_city_map/)**

## Features

- **Procedural Road Generation**: Organic branching, Grid, and Radial patterns.
- **Dynamic Terrain**: River, Lake, and Coastline generation with bridge building.
- **Zoning & Blocks**: Automatic detection of city blocks and filament suburbs.
- **Building Generation**: Recursive subdivision for dense city centers.
- **Interactive**: Pan, zoom, and customize generation parameters in real-time.

## Tech Stack

- React
- TypeScript
- Vite
- Canvas API
- Tailwind CSS

## Development

```bash
npm install
npm run dev
```
